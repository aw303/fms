
# Logistics & Fleet Management System

A modern Logistics & Fleet Management platform built with Laravel API and Angular Material.

## Features
- Authentication & RBAC
- Customer Management
- Vehicle Management
- Driver Management
- Booking Management
- Trip Management
- GPS Tracking
- Expense Management
- Invoice Management
- Dashboard & Analytics

## Technology Stack
### Backend
- Laravel 12
- PHP 8.3+
- MySQL/PostgreSQL
- Sanctum
- Spatie Permission
- Redis

### Frontend
- Angular 19+
- Angular Material
- TypeScript
- ApexCharts

## Docker Development
Docker support is included for the full local stack: Laravel/PHP, Angular, MySQL, PostgreSQL, Redis, and Nginx.

See [DOCKER_README.md](DOCKER_README.md) for setup and commands.
